# Leetcode solutions written in Javascript

## 分类标准

* 重点：必须掌握的题型。每一题都是频繁面试点，或者有着代表解法，或者可以举一反三。
* 提高：难度比较高的题，或者思路巧妙的题，常规面试一般不会出现，提升自我的目的可以练习
* 了解：面试基本不会出现，一般是解法很容易想到但是细节繁琐，或者和重点题型的解法重复
